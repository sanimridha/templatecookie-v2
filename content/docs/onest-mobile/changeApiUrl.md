#### Change api url

You have to go **src/config/index.js** and change

**ROOT_URL_SECURE**

**API_BASE_URL**

**BASE_URL**

```javascript
export const ROOT_URL_SECURE = "your root url source here";
export const API_BASE_URL = "your api base url here";
export const BASE_URL = "your base url here";
```
