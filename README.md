## Run Locally

Clone the project

```bash
  git clone https://github.com/zakirsoft/templatecookie-v2.git
```

Go to the project directory

```bash
  cd templatecookie-v2
```

Install dependencies

```bash
  npm install / yarn 
```

Start the server

```bash
  yarn dev
```
